# ContactsManager

Un programme Java simple pour gérer une liste de contacts.

## Ce que fait le programme

- Ajouter des contacts (nom + numéro de téléphone)
- Rechercher un contact par son nom

## Les fichiers

- `Contact.java` : représente un contact
- `ContactsManager.java` : gère la liste des contacts
- `Main.java` : lance le programme

## Comment l'utiliser

1. Ouvrir le projet dans IntelliJ
2. Lancer `Main.java`
3. Le programme affiche les contacts ajoutés et le résultat de la recherche

## Résultat attendu

```

Contact ajouté : Bob
Contact ajouté : Chloé

Contact ajouté : Emma

Nom : Alice

```
